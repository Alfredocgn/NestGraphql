# Dev

1. Clone the project
2. Copy ```env.template``` and rename to ```.env```
3. Execute ```pnpm install```
4. Run docker image ``` docker-compose up -d``` Docker desktop needed
5. Run proyect ```pnpm run start:dev```
6. Check site ```localhost:3000/graphql```
